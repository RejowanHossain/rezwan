Zelevent - Tech Summit Website

Introducing the new tech summit web design suite designed for Webflow, redesigned from the ground up to streamline creative workflow.

Goal

Meet Zelevent, a seamless and user-centric web template for both remote and hybrid event summit catered to both in-person and virtual audiences, with features like live streaming, online events and clear event agendas. The focus is on making navigation simple, keeping users engaged, and ensuring easy access to all the details for both virtual and physical attendees.

My Role Research, ideation, UI design

Problem

Existing competitors primarily cater to either online or offline events, lacking the ability to support hybrid experiences effectively. The challenge was to design a solution that balances both formats with a minimal design, streamlined user flows, and an uncluttered architecture for seamless navigation.

Target Audiences

  • Organizers of remote, hybrid, and online-only events
  • Attendees who will be participating both physically and virtually
  • Speakers and Sponsors looking for detailed event information and engagement opportunities

User Personas

Breakdown

The template could be dynamic based on the selection of the event and it's type. But it comes with some new drawbacks and a shaky user experience.

Solution

A centralized streaming page is used for both online and hybrid events. For offline events, it streams the event live, and for online events, the host runs sessions on the same page. Access is password-protected to ensure only ticket holders can join. After purchasing a ticket, attendees receive a link and a unique code via email. They use the link to land on the password page and then enter the code to unlock the streaming page, keeping it secure and exclusive.

User Flow

One of the primary goal was to create a simple user flow and so that users can escape the learning curve and get their desired outcome with their minimalistic approach.
